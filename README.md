#  Vehicle Listing App
 A web application built with Flask that allows users to browse, add, and manage vehicle listings. The application includes user authentication, image uploads, and a clean interface for managing vehicles.
## Features
- User Registration & Login
- Secure Authentication
- Add Vehicle Listings
- Upload Vehicle Images
- View All Listed Vehicles
- Responsive User Interface
- Database Integration
## Tech Stack
- Python 
- Flask
- HTML 
- CSS 
- JavaScript 
- Postgresql

## Installation Clone the repository 
```bash git clone https://github.com/abdul-moiz-khurram/vehicle-listing-app.git ``` Go to the project directory 
```bash cd vehicle-listing-app ``` 
Install dependencies 
```bash pip install -r requirements.txt ``` 
Run the application 
```bash python app.py ``` 
Open in your browser 
``` http://127.0.0.1:5000 ```
